Principais Comandos do Git

Configuração

git config --global user.name "Seu Nome"
git config --global user.email "seu-email@example.com"

Criação e Clonagem de Repositórios

git init
git clone URL_do_Repositório

Comandos Básicos

git status
git add nomedoarquivo
gir add .
git commit -m "mensagem do commit"
git log

Trabalhando com Branches

git branch
git checkout nomedobranch
git checkout -b novobranch
git merge nomedobranch
git branch -d nomedobranch

Atualizando e Enviando para o Repositório Remoto

git pull origin nomedobranch
git push origin nomedobranch

Desfazendo Alterações

git reset HEAD nomedoarquivo
git checkout nomedoarquivo

Revertendo Commits

git log
git revert HASH_DO_COMMIT

Ignorando Arquivos

# Crie um arquivo chamado .gitignore e adicione os nomes dos arquivos/diretórios que deseja ignorar.
# Exemplo:
logs/
node_modules/
secretfile.txt

Visualizando Diferenças

git diff
git diff nomedoarquivo
